# Headshots for the NYU Urology map

Drop each headshot here named EXACTLY as in HEADSHOT_FILENAMES.csv (lowercase, hyphenated, .jpg).
Example: Bofeng Chen -> img/bofeng-chen.jpg

The map auto-detects a photo: if img/<slug>.jpg loads, the marker becomes a circular headshot ringed in the person's subspecialty color; otherwise it stays a colored dot. No code change needed.

Square images, ~200x200px or larger, work best. Keep a copy of the live source URL as fallback if you prefer not to host locally.
